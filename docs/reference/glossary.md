# Glossary

Terms used in Playframe documentation.

| Term | Definition |
|------|------------|
| **Playframe** | This project — the game portfolio website (embeds Unity Play via iframe) |
| **Unity Play** | Unity's free WebGL game hosting platform at [play.unity.com](https://play.unity.com/) |
| **WebGL** | Web graphics API; Unity can export games to run in the browser |
| **Content Collection** | Astro feature for typed, validated markdown/MDX content |
| **Slug** | URL-safe identifier for a game, e.g. `neon-drift` → `/games/neon-drift` |
| **Share URL** | Unity Play page link (`play.unity.com/en/games/...`) |
| **Embed URL** | iframe `src` for embedding Unity Play (`play.unity3dusercontent.com/webgl/...`) — required for released games |
| **Vercel** | Static site hosting for Playframe (no WebGL files) |
| **Static site** | Pre-built HTML/CSS/JS served from a CDN; no server runtime |
| **Island** | Astro pattern for hydrating interactive components with minimal JS |
| **Build** | Unity's packaged, playable output for a target platform |
| **Thumbnail** | Cover image shown on game cards (recommended 16:9) |
